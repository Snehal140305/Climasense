# ClimaSense AI – Smart Weather Advisor

A real-time smart weather dashboard built using React and OpenWeather API.

## Features
- Live weather search by city
- Intelligent weather advisory system
- Mini temperature trend graph
- Temperature unit conversion (°C / °F)
- Recent search history
- Loader & error handling UI
- Responsive mobile-style layout

## Tech Stack
- React.js
- CSS3
- OpenWeather API
